Bubbling Up

Game Jam - UE5 Blueprints

Bubbling Up was made in 48 hours as part of the 2025 Global Game Jam. The theme was 'Bubbles', so we took this theme and created a platformer which utilises bubbles as objects to jump on.


My team used UE5 Blueprints to do this, I primarily focused on the texturing and VFX which brought the bubbles to life.

Details

  • Group Game Jam Project (4 team members)
  • Awarded best trailer within the university
  • Role: VFX artist and game trailer editor
  • Duration: 48 hours (January 2025)
  • Genre: 3D adventure platformer

Game Mechanics / Features

Game Key Feature: bubble platform mechanic. The player can shoot bubbles which will expand into solid platforms that can then be used to jump on
