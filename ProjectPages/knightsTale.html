A Knight's Tale

Solo University Project - Unity

This game is a prototype of a sword fighting tournament - one of the 3 events found in a classic knight tournament (Sword fighting, Archery, Jousting). Players fight their way through increasingly more difficult battles with enemy knights and aggressive gargoyles to claim the title of champion.


This project served as my introduction to Unity, the goal was to create a playable vertical slice in 9 weeks, so I opted to take a game I had been wanting to make, and and make a pixel art version of one part of it (the sword fighting section).

Details

  • Solo University Project
  • Grade: First
  • Role: Generalist Programmer (UI, Mechanics), Level Designer, 2D Artist
  • Duration: December 2024 - February 2025 (9 Weeks)
  • Genre: 2D, 3rd person Arena Combat game

Game Mechanics / Features

Key Responsibilities: created the pixel art background, knight-themed tile maps and the knight character with run animation. Developed simple opponent AI. Designed and implemented UI elements and created simple VFX.
